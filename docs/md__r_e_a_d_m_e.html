Installation:

Installing on an ECC Machine:

git clone https://github.com/derekdstratton/Best-C-Compiler
cmake Best-C-Compiler
make

Usage:

  • To run the Scanner/Parser, type ./ScannerParser
  • To run the Symbol Table driver, type ./SymbolTable
