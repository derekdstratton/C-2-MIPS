Three Address Code

The Three Address Code is generated by traversing the Abstract Syntax Tree and outputting intermediate code. This intermediate code is stored in a table of strings, with each command consisting of an instruction type (MINUS, BR, etc), 2 source arguments, and a destination argument. Most of the syntax follows the CS 460 Intermediate Language Specification, with some slight tweaks.

One major tweak is for functions, where we use a PUSHPARAM command.
